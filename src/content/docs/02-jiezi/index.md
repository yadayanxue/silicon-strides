# 卷二 · 芥子（裸机与实时）

> 纳须弥于芥子，掌控实时。

芥子虽小，可纳须弥。此卷探索资源受限的嵌入式世界——没有操作系统的庇护，每一字节内存、每一个时钟周期都需精打细算。从裸机启动到 RTOS 调度，从中断优先级到功耗优化，这是最贴近硬件的编程艺术。

## 章节导览

- [**裸机编程**](./01-bare-metal/)：启动代码、链接脚本、内存布局
- [**中断与异常**](./02-interrupts/)：向量表、ISR、中断嵌套与优先级
- [**RTOS 基础**](./03-rtos-fundamentals/)：任务调度、信号量、消息队列、内存池
- [**外设驱动**](./04-peripheral-drivers/)：GPIO、UART、SPI、I²C、DMA
- [**低功耗设计**](./05-low-power-design/)：睡眠模式、时钟门控、功耗状态机

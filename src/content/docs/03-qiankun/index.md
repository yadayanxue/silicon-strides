# 卷三 · 乾坤（系统与网络）

> 定乾坤，理乱象，调度万物。

乾坤者，天地之序。此卷聚焦操作系统内核与网络协议栈——前者管理硬件资源、调度万千进程；后者连接孤岛、编织通信之网。理解内核，方能理解一切上层软件的边界与可能。

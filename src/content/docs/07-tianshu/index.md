# 卷七 · 天枢（安全与御道）

> 执天枢，御外魔，坚不可摧。

天枢者，北斗之柄，执之可御天下。此卷从密码学的数学根基走向系统安全的工程实践——对称加密与非对称加密如何守护数据，零知识证明如何实现不泄露信息的验证，攻防博弈如何在攻击面与可信基之间展开。

## 章节导览

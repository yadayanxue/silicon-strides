# 硅步千里 (Silicon Strides)

> 不积跬步，无以至千里。从一粒硅沙，到数字宇宙的漫游指南。

《硅步千里》是一套**全栈计算机科学技术参考文档**，以东方哲学的"八卷宗"为索引结构，覆盖半导体物理、体系架构、操作系统、数据库、图形学、AI、安全、软件工程等领域的核心概念。每一卷独立成篇，在概念交汇处交叉引用，供查阅而非线性阅读。

🌐 在线阅读：**[yadayanxue.github.io/silicon-strides](https://yadayanxue.github.io/silicon-strides/)**

## 📚 八卷宗

| 卷宗 | 领域 |
|------|------|
| 卷零 · 灵犀 | 数学、逻辑与算法理论 |
| 卷一 · 微尘 | 半导体物理、体系结构与指令集 |
| 卷二 · 芥子 | 嵌入式系统、RTOS 与裸机编程 |
| 卷三 · 乾坤 | 操作系统、内核与网络协议栈 |
| 卷四 · 渊海 | 数据库、分布式系统 |
| 卷五 · 万象 | 计算机图形学、前端与交互 |
| 卷六 · 须弥 | 机器学习、深度学习与语言模型 |
| 卷七 · 天枢 | 密码学与系统安全 |
| 卷八 · 千里 | 软件工程、DevOps 与架构道法 |

## 🚀 本地运行

```bash
git clone https://github.com/yadayanxue/silicon-strides.git
cd silicon-strides
npm install
npm run dev
```

访问 `http://localhost:4321`。
